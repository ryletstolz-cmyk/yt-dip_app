
YTMP3 / YTMP4 Downloader
1.download File.zip
2. open app File
3.unzip
HOW TO USE:

1. Open YTMP3_Launcher.cmd
2. Choose MP3 or MP4
3. Paste the YouTube URL
4. Click Download

FILES SAVE LOCATION:
- MP3 downloads → Audio\<Video or Playlist Name>\
- MP4 downloads → Video\<Video or Playlist Name>\

REQUIREMENTS:
- yt-dlp.exe must be in the same folder
- ffmpeg.exe must be in the same folder
